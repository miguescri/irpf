# IRPF

A simple Go CLI to estimate the net salary in Spain given a raw annual amount.

**THIS IS NOT A FINANCIAL TOOL, USE AT YOUR OWN PERIL**

Example:

```shell
$ go build
$ ./irpf -a 28000
Raw income: 28000 €
Retention: 6565.5 €
Net income: 21434.5 €
Net monthly income: 1786.2083333333333 €
```
